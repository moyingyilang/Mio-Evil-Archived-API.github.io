Mio 邪恶档案随机图片 API

一个简单的服务，每次请求随机返回一张来自 Mio 邪恶档案的图片

API 测试

点击下方按钮测试随机图片 API：

点击测试按钮获取随机图片

API 接口

使用以下 URL 获取随机图片：

https://moyingyilang.github.io/Mio-Evil-Archived-API.github.io/api/random.jpg

API 直接返回一张 JPG 图片，您可以在任何支持图片显示的地方使用此链接。

使用示例

以下是在不同场景中使用此 API 的示例：

HTML 图片

直接在 <img> 标签中使用：

<img src="https://moyingyilang.github.io/Mio-Evil-Archived-API.github.io/api/random.jpg" alt="随机图片">

Markdown 文档

在 README 或文档中使用：

![随机图片](https://moyingyilang.github.io/Mio-Evil-Archived-API.github.io/api/random.jpg)

命令行 (curl)

使用 curl 下载随机图片：

curl -o random.jpg \ https://moyingyilang.github.io/Mio-Evil-Archived-API.github.io/api/random.jpg
